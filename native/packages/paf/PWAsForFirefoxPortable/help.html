PWAsForFirefox Portable Help

A tool to install, manage and use Progressive Web Apps (PWAs) in Mozilla Firefox

PWAsForFirefox Portable is a tool to install, manage and use Progressive Web Apps (PWAs) in Mozilla Firefox. It's packaged with a PortableApps.com launcher as a portable app, so you can use it from a cloud folder, portable drive, or local folder without needing to install it on each PC. Usage

The portable installer for PWAsForFirefox Portable can be downloaded from the GitHub Releases page of the project. It is recommended to install it through the PortableApps.com Platform for additional integration with its app menu.

Before being able to use the browser extension with the portable version, you have to start PWAsForFirefox Portable. This will launch a small background program that makes sure the browser is connected to the portable version. You can close this program from the taskbar tray icon.

When using with the PortableApps.com Platform, installed web apps will be automatically added to its app menu.

Known Problems

  • Adding web apps to start menu may not work.
  • Pinning web apps to the taskbar may not work.
